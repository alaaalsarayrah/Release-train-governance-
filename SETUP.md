# Setup

## Project
- Name: `thesis-agentic-scrum`
- Path: `D:/Project`

## Requirements
- Node.js 18+
- npm

## Install
```powershell
cd D:\Project
npm install
```

## Run (Development)
```powershell
npm run dev
```

## Build
```powershell
npm run build
```

## Start (Production)
```powershell
npm run start
```

## Utility Commands
```powershell
npm run smoke:planning
npm run smoke:ado
npm run generate:full-doc
npm run generate:project-doc
npm run generate:all-docs
```
